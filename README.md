# AI-Business-Intelligence-Platform
An end-to-end AI Business Intelligence platform that transforms Excel data into interactive dashboards and AI-powered business insights using Python, SQL, Power BI, and Large Language Models.

## 🎯 Project Goal

The objective of this project is to build a complete AI-powered Business Intelligence Platform capable of:

- Cleaning raw business data
- Performing exploratory data analysis
- Generating business visualizations
- Querying data using SQL
- Building interactive Power BI dashboards
- Integrating an AI assistant using Large Language Models and Retrieval-Augmented Generation (RAG)
